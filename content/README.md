# Blog Content

This directory contains blog posts in MDX format.

## Structure

```
content/
└── blog/
    ├── no/           # Norwegian posts
    │   └── example.mdx
    └── en/           # English posts
        └── example.mdx
```

## Creating a new post

Create a new `.mdx` file in the appropriate language folder with the following frontmatter:

```mdx
---
title: "Your Post Title"
description: "A brief description of your post"
date: "2026-01-08"
---

Your content here...
```

## Example

```mdx
---
title: "Hvordan bygge effektive utviklerteam"
description: "Mine erfaringer med teamledelse i teknologibransjen"
date: "2026-01-08"
---

# Hvordan bygge effektive utviklerteam

Innhold her...
```
